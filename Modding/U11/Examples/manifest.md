# Manifest file

This manifest file is required for mods to load.
The GameVersion needs to at **least** match the version of the game you are modding for

```json
{
    "Name": "MyMod",
    "Description": "This is my cool mod",
    "Author": "Wully",
    "ModVersion": "1.0.0",
    "GameVersion": "11.0"
}
```
